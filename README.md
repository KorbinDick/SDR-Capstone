# SDR-Capstone
Software Defined Radio Capstone Project / York College of Pennsylvania
-----------------------------------------------------------------------
## ESP32_WROOM-32 based software defined radio project using Pmod I2S2 ADC/DAC, Direct Digital Synthesis (sine LUT and phase accumualtor), Codec2, adjustable framing bits, and I/Q signals.
-----------------------------------------------------------------------
Included Libraries:
https://github.com/sh123/esp32_codec2_arduino/tree/master
